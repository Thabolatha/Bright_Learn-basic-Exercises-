# 📘 SQL Exercises

## 📌 Overview

This folder contains SQL exercises designed to build and strengthen foundational data analytics skills. The exercises progress from basic SQL queries to more advanced concepts such as joins and conditional logic using CASE statements.

---

## 📂 Exercises Breakdown

### 🟢 Exercise 1: Basic SQL Queries

Focus:

* SELECT statements
* Filtering data using WHERE
* Sorting using ORDER BY

---

### 🟡 Exercise 2: Aggregations

Focus:

* SUM, COUNT, AVG
* GROUP BY
* Basic data summarization

---

### 🔵 Exercise 3: Joins

Focus:

* INNER JOIN
* Combining multiple tables
* Understanding relationships between datasets

---

### 🟣 Exercise 4: Case Statements & Time Analysis

Focus:

* Using CASE statements for classification
* Creating new columns (e.g., time buckets)
* Categorizing data (e.g., weekday vs weekend)

---

These exercises are part of my SQL learning journey, aimed at improving my ability to:

* Perform data transformations
* Analyze datasets for business insights
